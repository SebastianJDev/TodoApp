# To-Do App - Sistema de Gestion de Tareas

Aplicacion web para crear, editar, eliminar y completar tareas, organizadas por estados (Pendiente, En proceso, Completada).

## Tecnologias
- JavaScript
- Node.js + Express
- SQLite (better-sqlite3)
- HTML/CSS/JS
- Vitest + Supertest

## Requisitos
- Node.js 20+

## Ejecucion
1. Instalar dependencias:
   npm install
2. Iniciar la app:
   npm start
3. Abrir en navegador:
   http://localhost:3000

## Pruebas
Ejecutar tests:

npm test

## Funcionalidades implementadas
- Crear tareas con validacion
- Editar tareas
- Eliminar tareas
- Marcar tareas como completadas
- Organizacion visual por estado
